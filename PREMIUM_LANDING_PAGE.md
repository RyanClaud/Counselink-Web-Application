## Design Features

### Color Palette
```css
Primary Gradient: #667eea → #764ba2 (Purple)
Secondary Gradient: #f093fb → #f5576c (Pink)
Accent Gradient: #4facfe → #00f2fe (Blue)
```

### Typography
- **Headings**: Playfair Display (Serif, elegant)
- **Body**: Inter (Sans-serif, modern)
- **Weights**: 300-800 for hierarchy

### Animations
- **Fade-in on scroll**: Intersection Observer API
- **Floating hero image**: CSS keyframes
- **Hover effects**: Transform and shadow transitions
- **Smooth scrolling**: Native smooth scroll

### Responsive Design
- **Mobile-first** approach
- **Breakpoints**: 768px, 992px, 1200px
- **Flexible grids**: Bootstrap 5
- **Touch-friendly**: Large buttons and spacing
